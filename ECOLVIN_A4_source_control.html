SOURCE CONTROL

Source control plays a key role in all software development as it allows multiple areas of code to be worked on simulatiously, without interfering with each others work. It also helps protect the working source code as people don't work direclty on it, and it also facilitates the ability to work on code on different computers that are not otherwise connected. I have experience using git based source control software, to manage the development of software solutions in a way that keeps the source safe and allowing colabortative work.
